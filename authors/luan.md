# 乱·宠魅型

> 作者风格定义文件。项目 `rules.md` 通过引用此文件获取风格锚点、样本和校准清单。
> 项目可在引用基础上追加覆盖项（见 rules.md 第 8 节）。
> 
> **基础作品**: 《宠魅》by 乱
> **风格核心**: 系统化世界观 + 回合制战斗 + 量化压力传达 + 信息密集对话

---

## 1. 风格锚点

### 1.1 作者与作品定位

| 作者 | 代表作品 | 核心风格特征 | 典型段落类型 |
|------|---------|------------|-------------|
| **乱** | 《宠魅》 | 系统化世界观构建、回合制战斗描写、量化压力传达、信息密集对话、成长量化路径 | 战斗回合、设定层级、量化压力、训练步骤 |

### 1.2 风格流派与定位

- 风格流派：玄幻网文·成长竞技型
- 在流派中的位置：系统设定严密、成长线扎实的一档，介于硬核设定型与快节奏爽文型之间
- 与相邻流派的边界：比纯爽文更注重设定逻辑，比纯硬核玄幻更注重读者代入感和节奏流畅

### 1.3 风格关键词

- 核心词（5 个）：**系统设定、回合战斗、量化成长、羁绊默契、压力传导**
- 质感描述：设定如表格清晰，战斗如回合分明，成长如阶梯攀升，数字如锚点稳定

### 1.4 反面风格（绝对避免）

| 反面倾向 | 具体表现 | 替代方案 |
|---------|---------|---------|
| 设定空洞 | 世界观只有名字没有机制，能力只有称号没有效果 | 建立完整的等级体系、能力机制、成长路径 |
| 成长跳跃 | 主角能力突然提升没有过程，境界跨越没有训练 | 每次提升必须有训练/战斗/领悟的中间步骤 |
| 战斗模糊 | 打斗只写"激烈""惊险"，没有具体技能和回合 | 每个战斗回合要写出具体技能名称、效果、结果 |
| 情绪标签堆砌 | 连续使用"愤怒""恐惧""震惊"等直接标签 | 用动作（握拳/冷汗/瞳孔放大）和环境（气氛/声音）传达 |
| 对话无信息 | 对话只有情绪表达没有实质内容 | 每句对话应包含信息、指令、判断或人物性格体现 |

### 1.5 叙事节奏特征

- **战斗节奏**：技能名（1字）→ 施展过程（5-15字）→ 效果描述（5-10字）→ 结果定格（短句），形成清晰回合
- **设定节奏**：种族分类 → 特性说明 → 具体数值 → 市场价值，信息层次分明
- **成长节奏**：当前状态（阶段/等级）→ 训练过程 → 效果量化 → 新状态确认，形成闭环
- **对话节奏**：时间+地点+对手+赌注+条件，一次说完不分散

### 1.6 情感表达方式

- 情感传达路径：情绪→动作/生理反应→环境氛围→读者感受，允许适度直接标签但不滥用
- 克制度：在紧张战斗场景保持克制，在成长突破、羁绊加深场景可以适度释放
- 爆发时机：主角突破瓶颈、战胜强敌、获得关键伙伴、逆境反转时允许直接情绪描写

### 1.7 意象与感官偏好

- **战斗意象**：妖火邪焰（赤红）、魔焰（深蓝）、雷电（紫色）、冰霜（雪白）、鲜血（嫣红）
- **魂宠特征**：银白毛发、六尾/多尾、焰蹄、邪眸、翅膀状态
- **数值锚点**：阶段（五段七阶）、等级（高等战将级）、防御级别（三级初期）、技能等级（后期/全期）
- 意象使用规则：意象必须与具体能力对应，禁止装饰性描写

### 1.8 留白策略

- **必须写**：技能名称和等级、战斗回合过程、成长训练步骤、价值量化数字
- **绝不写**：无关的路人心理、没有信息量的闲聊、模糊的情绪标签（"愤怒""恐惧"）
- **留白位置**：战斗结束后的情绪回味、成长突破后的感慨、章节末尾的悬念

### 1.9 量化系统规则

> 乱的风格核心：一切皆可量化，数字是设定锚点。

| 量化类型 | 典型表达 | 示例 |
|---------|---------|-----|
| **等级量化** | 种族等级+阶段 | 高等战将级、五段七阶 |
| **技能量化** | 技能等级 | 初期/中期/后期/全期 |
| **防御量化** | 防御级别 | 三级初期防御皮毛 |
| **价值量化** | 金币价格 | 六级双属性魂晶七千金币 |
| **时间量化** | 成长周期 | 十天康复、四天药剂加速 |
| **概率量化** | 生存率 | 三千活一人 |

### 1.10 战斗心理传导

> 战斗不仅是技能对抗，更是意志和气势的压制。

- **气势释放**：妖气/魔焰/血气释放 → 周围环境影响 → 敌方心理压力
- **技能压制**：邪眸凝视/妖慑/梦碎之噩临 → 精神层面攻击 → 敌方意志崩溃
- **节奏控制**：先施压 → 后攻击 → 乘胜追击 → 不给喘息机会
- **杀意传达**：眼神变化（银色/赤红）→ 杀气释放 → 敌方战意瓦解

## 2. 风格样本

> 以下段落定义了本风格追求的文字质感。润色时以此为校准基准。
> 每个样本附带"锚点解析"，说明该样本为什么代表本风格。

### 样本 A：战斗场景的回合感

> 嗖嗖嗖！
> 银色的光芒一闪即逝，阴影之中，莫邪的速度瞬间增长，竟然直接从岩怪的身躯下穿过！
> 暗袭！马志立刻露出错愕之色，这种在阴影之下瞬间加速的技能正是许多兽属性魂宠所拥有的偷袭技能！
> 嘣！岩怪轰然倒地，泥土草根飞溅，周围土地也是一阵摇晃。

**锚点解析**：
- 节奏特征：短句开场的动作（嗖嗖嗖），中句描写过程，结尾短句定格结果
- 技能标注：明确写出技能名称（暗袭），读者能理解机制
- 战斗回合感：一个技能一个结果，形成清晰回合
- 与普通写法的差异：普通写法可能只写"莫邪突然加速穿过"，缺乏技能名称和回合结构

### 样本 B：成长突破的情绪

> 嘣！
> 獠狼狠狠的撞击在焰尾的腹部，焰尾发出了一声凄厉的惨叫声，被硬生生的撞飞了好几米，摩擦着地面滚动着，口中喷出了嫣红的鲜血。
> 尾巴上的火焰开始变得微弱，就像烛火一般，微风一过便会熄灭，九阶的焰尾蜷缩着身体，那双眼睛不甘的注视着獠狼和钢牙……

**锚点解析**：
- 情绪传达路径：伤痛（惨叫/流血）→虚弱（火焰微弱）→不甘（眼睛注视），层层递进
- 克制手法：不直接写"焰尾绝望了"，用火焰状态和眼神传达
- 留白位置：结尾的"不甘注视"给读者想象空间
- 与普通写法的差异：普通写法可能直接写"焰尾被打败了，十分绝望"

### 样本 C：环境与设定的结合

> 荆棘妖：植物界——藤系——荆棘妖族。
> 荆棘妖生长在有荆棘的丛林中，专门捕食途径荆棘林的生物，属于植物界中比较常见的食肉型妖物！
> 楚暮知道，自己一旦被拖到荆棘妖的本体位置的时候，身躯就会被分解，作为荆棘妖的肥料！

**锚点解析**：
- 设定嵌入：先写种族分类，再写特性，最后写威胁，信息层次清晰
- 环境服务剧情：荆棘林不是风景，是危险来源
- 与普通写法的差异：普通写法可能只写"林子里有危险植物"，缺乏设定体系

### 样本 D：对话的信息密度

> "你的月光狐需要战斗，两天后，就在这里。我的月光狐和你的十足蜈蚣，你赢了，我把我手中的一级虫属性魂核给你，你输了的话，就把你手中拿着的那块棕色兽属性魂核给我。如何？"

**锚点解析**：
- 对话节奏：单句长对话，信息密集（时间/地点/对手/赌注/条件）
- 信息量：包含约战约定全部要素，没有废话
- 与普通写法的差异：普通写法可能分散成多句"我们比试吧""赌注是什么""好就这样"

### 样本 E：训练过程的步骤感

> 小莫邪咬着幼牙，开始忍着爪子磨破的痛苦，连续攻击着树木。
> 唰唰唰！幼爪交叉划过，树皮立刻就被撕开了，一些木屑溅洒出。
> 继续。楚暮说道。
> 唰！唰！唰！小莫邪连续三次左右爪击，这直径十公分的树上出现了比较深的痕迹！
> 再继续，把这棵树撕断为止。楚暮继续说道。

**锚点解析**：
- 转场手法：从训练开始→动作描写→指令→继续→达成目标
- 信息压缩：每个"继续"代表多次重复，用结果而非过程展现
- 节奏变化：从详细描写开始，逐渐加速到结果呈现

### 样本 F：逆境压力的传达

> 这种死亡胁迫，只能逼得楚暮以最大的潜能提升自己的实力！
> 白魇魔的成长简直是一个杀戮的过程，往往是需要上千个资质比较好的寄体，才能够供养得起……
> 这位夏大人的估计，白魇魔成长到第三阶的时候，楚暮的时辰也差不多到了。

**锚点解析**：
- 爆发前铺垫：先写压力来源（白魇魔），再写具体威胁（第三阶），形成紧迫感
- 爆发手法：用数字（上千个寄体）和时间节点（第三阶）量化压力
- 与日常段落的反差：设定说明段落，但情绪压力渗透其中

### 样本 G：羁绊建立的温情

> 我记得有柄叫做莫邪的武器，静时温柔美丽，动时锐利邪异，就叫你莫邪吧，很适合你这个狡猾的小家伙。
> 呜～！好像不太喜欢楚暮将它说成是狡猾，发出了一声幽怨的叫声。
> 恩，是睿智……

**锚点解析**：
- 静默中的信息量：命名过程体现主角性格（用剑名命名魂宠）
- 节奏特征：对话→反应→修正，轻松互动节奏
- 情绪底色：不是激烈战斗，而是日常温情，体现羁绊建立

### 样本 H：设定体系的层级展示

> 魂宠的实力的提升，一般通过五个方面：
> 魂宠自身的阶段的提升。魂宠本身的磨练。魂宠师对魂宠的魂核喂养。药剂的强化，以及魂晶的淬炼。
> 其中最主要的自然是魂宠本身的磨练……

**锚点解析**：
- 时间切换手法：用列表形式展示设定体系
- 与主叙事的差异：脱离当前情节，进入设定说明模式
- 信息释放节奏：先列出所有方面，再指出重点，分层清晰

### 样本 I：量化成长确认

> 楚暮有些莫名的看着这价值不菲的四级双属性魂晶，露出了疑惑的神情。
> "这是夏大人赐你的。"之前站在夏广寒身后的那位冰霜女子枫香淡淡的说道。
> **锚点解析**：
- 价值锚点：四级双属性魂晶（量化）
- 情感克制：不写"惊喜""激动"，用疑惑神情传达
- 信息压缩：对话简短，价值暗示关系层级

### 样本 J：气势压制与战意瓦解

> 楚暮扫了一眼已经意志全无的苏宇，浮起了邪异的笑容，却是缓缓的念起了咒语，将冰空精灵收回到魂宠空间之中。
> 见苏宇已经没有半点战斗意思了，楚暮这才念起咒语，将狂妄无比的莫邪也收回到魂宠空间之中。

**锚点解析**：
- 战意判定："意志全无""没有半点战斗意思"（心理状态量化）
- 节奏控制：压制后不急于攻击，先收回魂宠（从容感）
- 杀意传达："邪异的笑容"代替"残忍""冷血"

### 样本 K：羁绊建立的默契

> 楚暮扫了一眼那渐渐消失在夜空中的青色身影，心中竟然升起一丝莫名的伤感……
> 青色的身影渐渐的消失在夜空中，虽然不知道天苍青蛰龙究竟要去哪里，但是楚暮感觉小家伙身上一定藏有很多辛酸的故事，否则它的眼神不会那么疲惫沧桑……

**锚点解析**：
- 情感传达路径：身影消失 → 莫名伤感 → 推测原因 → 眼神疲惫（克制）
- 留白手法：不写"悲伤""不舍"，用"莫名伤感"和推测传达
- 羁绊深度：能读懂对方眼神的疲惫，默契已建立

### 样本 L：战斗回合的连续性

> 瞬闪！
> 仿佛瞬间移动一般，莫邪几乎瞬间出现在了鲜血兽的面前，犹如一位喜欢艺术的暗杀者，锋利的爪刃根本不用使用任何技能便轻易的划开了鲜血兽五级的防御皮肤！
> 连续几次瞬闪，鲜血兽根本没有施展任何技能，身上已经多处了数条伤口！

**锚点解析**：
- 回合节奏：技能名 → 过程 → 结果，一个回合完整
- 连续攻击：不写"激烈战斗"，用"连续几次瞬闪"和具体伤口数量
- 量化防御："五级的防御皮肤"被轻易划开（实力差距量化）

## 3. 风格校准清单

> 每次润色时逐条检查，全部通过才算完成。
> 校准方式：从润色后正文中选取与风格样本 A-H 对应的段落，与样本并列对比，标注差异。

### 3.1 语言层面

- [ ] 设定解释是否有量化数据？（等级/阶段/防御级别/价值价格）
- [ ] 战斗描写是否有回合结构？（技能名→过程→结果，每回合清晰）
- [ ] 技能是否有等级标注？（初期/中期/后期/全期）
- [ ] 是否存在模糊的情绪标签？（"愤怒""恐惧""震惊"→改为动作/眼神）
- [ ] 形容词是否服务于量化？（"强大"改为"五段战将级"）

### 3.2 战斗层面

- [ ] 战斗是否有气势压制环节？（妖气释放→环境影响→敌方压力）
- [ ] 是否有精神层面技能？（邪眸凝视/妖慑等心理攻击）
- [ ] 战意判定是否清晰？（"意志全无""战意崩溃"等状态词）
- [ ] 连续攻击是否有回合计数？（"连续几次""数条伤口"）
- [ ] 防御差距是否量化？（"三级防御被轻易破开"）

### 3.3 成长层面

- [ ] 成长是否有量化路径？（当前阶段→训练→新阶段确认）
- [ ] 资源消耗是否有数字？（魂晶价格、药剂价值）
- [ ] 时间周期是否有标注？（十天康复、四年成长）
- [ ] 实力差距是否用等级对比？（"相差一段三阶"）

### 3.4 对话层面

- [ ] 对话是否信息密集？（时间+地点+对手+条件一次说完）
- [ ] 是否有明确的价值锚点？（金币/魂晶等级在对话中提及）
- [ ] 是否避免情绪废话？（不说"我很愤怒"，说"你死定了"）

### 3.5 羁绊层面

- [ ] 羁绊建立是否有互动细节？（命名、默契、眼神理解）
- [ ] 是否有克制传达？（不写"深厚感情"，用具体行为）
- [ ] 是否有训练/战斗中的配合描述？

### 3.6 样本对比校准

| 段落类型 | 对比样本 | 核心检验点 |
|---------|---------|----------|
| 战斗回合 | A、L | 技能名+等级+过程+结果 |
| 成长确认 | B、I | 量化数字+克制情感 |
| 设定层级 | C、H | 分类列表+重点标注+价值锚点 |
| 信息对话 | D | 时间+地点+对手+条件，一次说完 |
| 训练步骤 | E | 过程压缩+结果呈现 |
| 压力量化 | F | 数字威胁+时间节点 |
| 羁绊建立 | G、K | 命名+默契+眼神理解 |
| 气势压制 | J | 战意判定+节奏控制 |

## 4. 风格速查表

> 写作时快速参考，确保核心风格一致。

### 4.1 必用元素

| 元素类型 | 必须出现 |
|---------|---------|
| 战斗技能 | 技能名+等级（邪焰爪-后期） |
| 魂宠状态 | 阶段+等级（五段七阶-高等战将级） |
| 防御描述 | 具体级别（三级初期防御） |
| 价值参考 | 金币数字或对比（价值五万金币） |
| 时间周期 | 具体天数（十天康复） |

### 4.2 禁用写法

| 禁用类型 | 错误示例 | 正确替代 |
|---------|---------|---------|
| 模糊情绪标签 | "愤怒""恐惧""震惊" | 拳头握紧/冷汗流下/瞳孔放大 |
| 模糊战斗词 | "激烈战斗""惊险" | 具体技能名+回合结构 |
| 装饰性形容词 | "华丽""美丽""壮观" | 删除或改为具体特征描述 |
| 无信息对话 | "好的""明白了""行" | 删除或改为指令/判断 |
| 跳跃式成长 | "实力大增""变强了" | 具体阶段变化（五段→六段） |
