## 3. 风格校准清单

> 每次润色时逐条检查，全部通过才算完成。
> 校准方式：从润色后正文中选取与风格样本 A-H 对应的段落，与样本并列对比，标注差异。

### 3.1 语言层面

- [ ] 句子平均长度是否控制在目标范围内？（目标：XX 字以内，对话除外）
- [ ] 是否有超过 N 句的连续心理描写？（上限：N 句）
- [ ] 形容词是否超过必要量？（上限：每段不超过 N 个修饰性形容词）
- [ ] 是否存在廉价转折词？（"突然""忽然""猛地"等——替换为具体动作）
- [ ] 段落结尾是否留有呼吸空间？（避免每段都塞满信息）

### 3.2 情感层面

- [ ] 情绪是否靠动作/物件传达而非直接陈述？（"他攥紧拳头"而非"他很愤怒"）
- [ ] 克制度是否达标？（直接情绪标签不超过每千字 N 个）
- [ ] 爆发点是否克制使用？（全文直接情绪表达不超过 N 处）

### 3.3 对话层面

- [ ] 对话是否每句不超过目标字数？（上限：XX 字，长对话拆分）
- [ ] 对话是否有潜台词？（表面说的和实际意思之间有距离）
- [ ] 对话标签是否极简？（避免"他愤怒地说"等情绪标签）

### 3.4 意象与环境层面

- [ ] 环境描写是否服务于情绪而非单纯铺陈？
- [ ] 意象是否来自偏好列表？（避免与本风格无关的意象）
- [ ] 感官调用是否符合优先级？

### 3.5 样本对比校准

- [ ] 动作段 vs 样本 A：节奏/断句/力度是否一致？
- [ ] 情绪段 vs 样本 B：是否靠物件/动作传达而非直接陈述？
- [ ] 环境段 vs 样本 C：质感/渗透感是否到位？
- [ ] 对话段 vs 样本 D：极简/留白/潜台词是否到位？
- [ ] 过渡段 vs 样本 E：转场是否干净利落？
- [ ] 高潮段 vs 样本 F：爆发是否有足够的日常铺垫反差？
- [ ] 静默段 vs 样本 G：静默中是否有足够的信息量？
- [ ] 回忆段 vs 样本 H：时间切换手法是否与主叙事有区分？
