# 润色章节技能

这个技能用于润色小说章节，使其符合人类阅读习惯、符合小说写作习惯，避免生硬木讷的表达。

## 核心目标

1. **符合人类阅读习惯**：让读者能够自然流畅地阅读，不被生硬的表达打断
2. **符合小说写作习惯**：遵循小说创作的艺术规律，注重叙事技巧
3. **避免生硬木讷**：删除机械化的描述，增加生动性和感染力

## 触发条件

当用户说以下内容时，自动调用此技能：
- "润色第X章"
- "优化第X章"
- "润色章节"
- "优化章节"
- "使第X章符合人类阅读习惯"
- "符合小说写作习惯"
- "不要生硬木讷"

## 润色流程

### 第零步：加载作者风格约束

润色前必须先加载风格约束：
1. 读取 `{项目}/rules.md` §7 获取作者风格文件，提取：反面风格表、情感克制度、对话规则、留白策略、校准清单
2. 读取 `{项目}/rules.md` §8 项目级覆盖项（如有）
3. 合成**润色约束表**（情感密度/对话句长/形容词限制/句子长度/语气词），后续步骤以此表为最高优先级

**优先级规则**：
- 第一步至第三步（量化/技术描述/分隔符）为通用机械修复，始终执行
- 第四步至第九步受风格约束控制，约束表优先
- 第十步（AI高频词）：黑名单词如果出现在**作者风格样本中**则保留（风格优先），否则替换
- 第十一步（拟人化）不受风格约束影响，始终执行

### 第一步：删除量化
1. 搜索"约XX"模式
2. 删除或修改所有过于精确的量化：
   - 距离量化：约XX米、约XX厘米、约XX寸、约XX尺
   - 时间量化：约XX秒、约XX分钟、约XX小时
   - 角度量化：约XX度
   - 年龄量化：约XX岁
   - 数量量化：约XX人、约XX户、约XX箱

**示例：**
- ❌ `约五百米的树林` → ✅ `树林里`
- ❌ `约三小时后` → ✅ `傍晚时分`
- ❌ `约三十岁` → ✅ 删除年龄描述

### 第二步：删除技术化描述
1. 搜索以下模式并删除：
   - `**回合X：XX vs XX**`
   - `**场景X：XX**`
   - `**契约差异·XX对比**`
   - `**东侧切入成功。三名成员失去战斗能力。**`（粗体标记的战斗结果）
   - `"第X章"`（元信息引用）

**示例：**
- ❌ `**场景四：北侧埋伏者加入战斗**` → ✅ 删除
- ❌ `**契约差异·保护对比**` → ✅ 删除
- ❌ `"第十四章风刃·初期"` → ✅ `"风刃·初期训练时"`
- ❌ `最后一次见阿木，是第十七章突袭据点` → ✅ `最后一次见阿木，是突袭据点那次`

### 第三步：统一场景分隔符
1. 搜索`***`模式
2. 替换为`---`

**示例：**
- ❌ `***` → ✅ `---`

### 第四步：优化对话（风格感知版）

**先检查润色约束表中的对话规则**：

- **信息密集风格**（如 luan.md 样本D：对话承载设定信息）→ 不添加语气词，保持信息密度，对话优化方向为"增加信息量"而非"增加口语化"
- **极简留白风格**（如 cold_hard_school.md 样本D：断句节奏、每句≤10字）→ 不添加连接词，保持断句节奏，对话优化方向为"删除多余字词"而非"添加"
- **无特殊要求** → 按下方通用规则执行

**通用规则（仅在风格约束允许时执行）**：
1. 检查对话是否生硬
2. 添加口语化表达（仅限风格约束未禁止语气词时）：
   - 语气词："吗"、"呢"、"吧"
   - 连接词："但"、"所以"、"因为"
   - 主语："他"、"她"、"它"
3. 删除生硬的句号，改为逗号或问号（仅限风格约束允许时）

**示例：**
- ❌ `"伤口还好？"` → ✅ `"伤口还好吗？"`
- ❌ `"他们会来？"` → ✅ `"他们会来吗？"`
- ❌ `"来。"` → ✅ `"来吧。"`
- ❌ `"开始。"` → ✅ `"开始吧。"`
- ❌ `"你伤比我深。好好休息。"` → ✅ `"你伤比我深，好好休息。"`
- ❌ `"隐隐作痛。不影响活动。"` → ✅ `"隐隐作痛，但不影响活动。"`
- ❌ `"猎魂团精锐队长。"` → ✅ `"他是猎魂团精锐队长。"`
- ❌ `"老佣兵。基础扎实。"` → ✅ `"老佣兵了，基础很扎实。"`

### 第五步：优化思考部分
1. 检查思考部分是否重复引用对话
2. 删除引号，让思考更像内心独白
3. 简化重复内容

**示例：**
- ❌ 
  ```
  林默看着帐篷顶部的布，脑子里想着卡尔的话。
  
  "影猫的眼神空洞，服从命令，没有判断。"
  
  "你的星尘主动保护，有选择。"
  ```
- ✅ 
  ```
  林默看着帐篷顶部的布，脑子里想着卡尔的话。
  
  影猫的眼神空洞，服从命令，没有判断。星尘主动保护，有选择。
  ```

### 第六步：增加感官描写（风格感知版）

1. **检查润色约束表中的感官优先级**：
   - 按作者风格的感官优先级顺序增加描写（如冷硬派：触觉 > 听觉 > 视觉 > 嗅觉）
   - 仅使用风格文件中的偏好意象或同质感意象
   - 禁止使用与风格无关的装饰性意象
2. 检查战斗场景
3. 增加声音、视觉、触觉、情感描写（按优先级顺序）

**示例：**
- ❌ `刀刃灰白色划过空气` → ✅ `刀刃灰白色，划过空气，带着破风声`
- ❌ `金属碰撞声响起` → ✅ `金属碰撞声响起，火花飞溅`

### 第七步：检查逻辑
1. 检查角色知识是否合理
2. 检查货币体系是否统一
3. 检查契约状态是否一致

### 第八步：检查重复
1. 检查章节内重复
2. 检查跨章节重复

### 第九步：检查符号标记
1. 检查符号形状描述是否一致
2. 检查符号使用场合是否恰当

### 第十步：替换AI高频词

AI写作有典型的"味儿"，以下词汇/句式高频出现时极易被平台判定为AI生成，必须逐一替换或删除：

**黑名单词汇**（出现即替换）：
| AI高频词 | 替换方式 |
|---------|---------|
| 不禁 | 删除，直接写动作 |
| 宛如/犹如 | 删除，用具体描写代替 |
| 眼底闪过一丝XX | 改为具体微表情或直接写动作 |
| 嘴角微微上扬 | 改为其他笑法或删除 |
| 深吸了一口气 | 改为具体动作（握拳/停顿/转头） |
| 心中暗自思忖 | 改为内心直接吐槽短句 |
| 目光如炬 | 删除 |
| 眉头微蹙 | 改为其他表情 |
| 喉结滚动 | 删除或改为吞咽动作 |
| 不由得/不由自主 | 删除 |
| 仿佛/似乎（高频） | 保留1-2次，其余删除 |
| 竟然（高频） | 保留1次，其余删除 |
| 缓缓（高频） | 保留1次，其余删除 |

**黑名单句式**（出现即重写）：
- "他/她不禁XX" → 直接写XX动作
- "心中暗自思忖：'XX'" → "XX。"（内心独白不加引导词）
- "眼底闪过一丝XX，随即消失" → 删除
- "嘴角微微上扬，露出一抹XX的笑" → 换一种笑法或不写笑

### 第十一步：拟人化处理（反AI检测）

清理完成后，文本仍可能因模式规律性被识别为 AI 生成。此步骤打破规律，注入人类写作特征。

**1. 打破重复句式**
- 扫描全文，找出出现 3 次以上的句式结构/动作描写/开头方式
- 每处用不同方式重写（换动词、换句式、换视角）

**2. 加"废笔"**
- 人类写作有与主线无关的闲笔：环境噪音、无关小动作、角色的随机习惯
- AI 写的每个细节都"有意义"，太干净
- 每 500 字插入 1-2 处无关紧要的细节（如"远处有人在喊什么"、"他踢了一脚地上的石子"、"空气里有股铁锈味"）

**3. 节奏打散**
- 连续 3 句以上长度相近（±5字）时，插入一句极短（2-3字）或极长（50+字）打破规律
- 避免整章句子长度呈正态分布
- 段落字数要有差异：有的段3行，有的段10行，不要每段都5-6行

**4. 删除废话和总结句**

判断标准：**推动剧情/塑造人物 = 有用细节，否则 = 废话**。

| 有用细节（保留） | 废话（删除） |
|----------------|------------|
| 揭示性格的习惯动作 | 与性格无关的机械动作 |
| 服务情绪的环境描写 | 装饰性的环境堆砌 |
| 推进对话的信息 | 不推进对话的寒暄 |
| 伏笔相关的感官细节 | 无伏笔功能的感官堆砌 |

具体删除：
- **总结句**：段末/章末的总结（"这一战让他成长了不少"），让读者自己感受
- **心理独白注水**：一个情绪写成3句话的，缩为1个动作
- **动作扩写**：喝水只写"喝了口水"，不写杯子材质/水温/喝水姿态（除非杯子/水温有剧情意义）

**5. 对话不完美化**（在第四步口语化基础上，进一步注入人味）
- 第四步已处理：口语化、语气词、连接词
- 此步补充：打断（"等等——你说什么"）、跑题、没说完（"其实我觉得……算了"）、重复对方的话（"死了？"）
- 检查：是否有连续3句以上对话都在"推进剧情"？如果是，插入1-2句不推进剧情的废话对话

**6. 情绪拉扯**
- 网文核心是"爽"和"情绪价值"，AI 写的冲突像流水账
- 打脸场景要有层层递进的压迫感，不能一笔带过
- 虐恋场景要有具体细节支撑，不能只写"他很痛苦"

## 注意事项

1. **保持剧情完整**：不改变大致剧情，不删除关键情节
2. **保持风格统一**：所有章节使用相同的润色标准
3. **保持情感真实**：增加情感描写，但不夸张
4. **保持节奏流畅**：删除冗余描述，增加感官描写
